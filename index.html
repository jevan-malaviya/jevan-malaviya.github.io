GitHub
Starred Repos Reader

This is a robust github repos reader built with Flutter and written in Dart. Features include user auth (utilizing the github API), pagination, responsive search, local caching, and webviews for individual repos. It uses riverpod for DI and state management, the Dio http client, freezed data-classes and union types, and more. Currently supports both Android and iOS platforms, with web coming soon.
